# student-id-card-generator
A modern web application for generating professional student ID cards with photo upload functionality. Features a beautiful glass-morphism UI with floating animations, form validation, live preview, and PDF download. Users can upload passport photos (via file upload or camera) and download centered, professional-looking ID cards.

✅ Photo Upload (File or Camera)<br>
✅ Glass-morphism UI with Animations<br>
✅ Form Validation & Live Preview<br>
✅ PDF Generation with jsPDF<br>
✅ Responsive Design<br>
✅ Drag & Drop Support<br>
✅ Professional ID Card Layout<br>

Generate student ID cards instantly with this modern web application.<br>
